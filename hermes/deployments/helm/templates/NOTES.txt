## First Steps

1. **Verify Deployment**
   ```bash
   kubectl get pods -l app.kubernetes.io/name={{ include "hermes-ai-assistant.name" . }} -n {{ .Release.Namespace }}
   ```

2. **Check Logs**
   ```bash
   kubectl logs -l app.kubernetes.io/name={{ include "hermes-ai-assistant.name" . }} -n {{ .Release.Namespace }}
   ```

3. **Test Health Endpoint**
   ```bash
   curl {{ if .Values.ingress.enabled }}{{ index .Values.ingress.hosts 0 "host" }}/api/orchestrator/status{{ else }}localhost:8080/api/orchestrator/status{{ end }}
   ```

4. **Create a Conversation**
   ```bash
   curl -X POST {{ if .Values.ingress.enabled }}{{ index .Values.ingress.hosts 0 "host" }}/api/orchestrator/conversation{{ else }}localhost:8080/api/orchestrator/conversation{{ end }} \
     -H "Content-Type: application/json" \
     -d '{"context":{"user_id":"test_user","session_type":"general"}}'
   ```
